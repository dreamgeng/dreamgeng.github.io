  1. 安装Hexo

至此，Hexo 的安装已经完成了，我们开始建站。

建站

  1. hexo init

  2. hexo g //生成静态网页

  3. hexo s//作用是启动服务器。这时候就可以用浏览器打开网址： http://localhost:4000/ 来进行预览了。

  4. 配置ssh

    • 执行下面命令生成SSH
    • 三次回车之后，可以生成id_rsa.pub文件，这里面就是SSH key的内容，然后使用vim编辑器打开这个文件
    • 之后把里面的内容都拷贝下来，打开github，点击右上角自己的头像，点击settings，再点击SSH，之后添加new ssh key，最后把复制的信息都粘贴进去，title随便写，最后输入以下命令判断SSH是否配置好：
    • 若出现以下问题：

    The authenticity of host ‘github.com (13.250.177.223)’ can’t be established.
    RSA key fingerprint is SHA256:nThbg6kXUpJWGl7E1IGOCspRomTxdCARLviKw6E5SY8.
    Are you sure you want to continue connecting (yes/no)?

    只需输入yes，然后回车即可；

    • 最后出现：Hi dreamgeng! You’ve successfully authenticated, but GitHub does not provide shell access.

    则表示已经配置好ssh。
