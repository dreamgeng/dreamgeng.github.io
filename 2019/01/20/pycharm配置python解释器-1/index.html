这个问题google了好久都没有找到正确的解决办法，因为明明已经安装了pyquery，但为什么导入不成功呢？而且在终端就可以导入成功，如下：

终端下：

然后我去查看了pycharm下的解释器：发现是python3.6，而终端下成功导入的是使用的python3，

于是我又启动python3.6交互界面，这时导入pyquery出现如下错误：

于是，问题确定下来，就是python解释器的问题。

那么，pycharm下如何配置python解释器呢？

  1. 在File目录下选择Setting，然后如图选择 Project Interpreter，如图：

此时，解释器为python3.6，然后点击右上角的下拉键，可以看到已经安装的python解释器，选择需要的即可：

但是，如果下拉键没有需要的怎么办呢？这时看到下拉键右边还有一个设置按钮，点击，add，弹出如下界面：

按图示选择，选择下拉键之后就会出现已经安装的解释器，导入即可。
